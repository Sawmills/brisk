# Brisk

Drop a folder, get a site.

Brisk is an open-source internal hosting platform inspired by Shopify's
[Quick](https://shopify.engineering/quick): upload a folder of HTML and assets,
get a live URL, and use a zero-config browser API for the backend things every
little site eventually wants — a database, file uploads, AI, realtime
websockets, and identity.

> Work in progress — full docs landing soon.
